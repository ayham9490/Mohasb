# Mohasb - تطبيق المحاسبة الذكي

تطبيق محاسبي متقدم لإدارة الحسابات والمعاملات المالية بسهولة وكفاءة.

## 🎯 المميزات الرئيسية

### 📊 لوحة التحكم
- عرض إحصائيات شاملة للحسابات
- عرض آخر المعاملات
- حساب الرصيد الصافي تلقائياً

### 💰 إدارة الحسابات
- إنشاء حسابات جديدة بأنواع مختلفة (عام، عميل، موردّ)
- حذف الحسابات غير المستخدمة
- عرض قائمة الحسابات النشطة

### 📝 تسجيل المعاملات
- تسجيل المدفوعات والمستحقات
- دعم عملات متعددة (SAR, USD, EUR, AED)
- إضافة بيان توضيحي لكل معاملة
- حذف المعاملات الخاطئة

### 📄 كشف الحساب
- عرض تفصيلي لجميع معاملات الحساب
- جدول محاسبي احترافي
- حساب الأرصدة والمجاميع تلقائياً
- تقسيم المعاملات حسب العملة

## 🛠️ المتطلبات التقنية

## 📦 التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone <repository-url>
cd mohasb-app
```

### 2. تثبيت المكتبات
```bash
pnpm install
```

### 3. تشغيل التطبيق محلياً
```bash
# لتشغيل على الويب
pnpm dev

# لتشغيل على Android
pnpm android

# لتشغيل على iOS
pnpm ios
```

### 4. بناء ملف APK
```bash
# استخدم لوحة التحكم في Manus
# انقر على زر "Publish" لبناء APK
```

## 🗄️ قاعدة البيانات

### جدول الحسابات (accounts)
```sql
- id: معرّف فريد
- userId: معرّف المستخدم
- accountId: معرّف الحساب
- accountName: اسم الحساب
- accountType: نوع الحساب
- createdAt: تاريخ الإنشاء
```

### جدول المعاملات (transactions)
```sql
- id: معرّف فريد
- userId: معرّف المستخدم
- transactionId: معرّف المعاملة
- transactionAccountId: معرّف الحساب المرتبط
- transactionType: نوع المعاملة (debit/credit)
- transactionAmount: المبلغ
- transactionCurrency: العملة
- transactionDescription: البيان
- transactionDate: تاريخ المعاملة
- createdAt: تاريخ الإنشاء
```
